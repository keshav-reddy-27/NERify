About the project!

NERify

NERify is a lightweight Named Entity Recognition (NER) web application that identifies entities such as Persons, Organizations, and Locations from input text.

Built using DistilBERT and fine-tuned on the CoNLL-2003 dataset, NERify offers real-time predictions through a simple and clean interface. The backend is powered by Flask, while the frontend uses Bootstrap for responsive design.

Whether you're exploring NLP or integrating NER into your projects, NERify provides a hands-on demo to get you started!
